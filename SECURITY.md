# Security Policy

## Supported Versions

Only the current exact numeric release tag receives fixes. This repository is
a tooling specification (docs, Brewfile, shell configs); operational safety
comes from explicit pinning.

| Version | Supported |
| --- | --- |
| Current exact tag `3.0.0` | yes |
| The `3.x` line label tracks only the latest released patch | no |
| Pre-3.0.0 history (fork-era tags `v1.x`/`v2.x`) | no |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Report privately via GitHub Security Advisories:

1. Open the [Security Advisories](https://github.com/rldyourmnd/awesome-terminal-for-ai/security/advisories) page of this repository.
2. Click "Report a vulnerability" and include reproduction details.

The parked `rldyourterm` engine submodule has its own repository and policy:
report engine issues at
[rldyourmnd/rldyourterm](https://github.com/rldyourmnd/rldyourterm/security/advisories).
